# FishAttack
Jogo desenvolvido como forma de avaliação da matéria de Linguagem de Programação II do curso técnico em Informática do IFAM.

## Comandos do jogo

| Comandos          	| Função            	|
|-------------------	|-------------------	|
| Tecla Espaço 	      |  Atirar 	          |
| Tecla Enter         | Reinicia o jogo   	|
| Setas direcionais   | Movimentação da nave|
